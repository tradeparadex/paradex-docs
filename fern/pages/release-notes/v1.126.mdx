      * TPSL Updates: Linked TPSL orders now automatically resize if the parent limit order size is modified.

      * Improve JWT verification performance with caching and optimize Starknet order message hashing for better concurrency and speed.
      * Sets a safety buffer for `RPI` orders to prevent them from crossing the book too much. RPI orders cannot cross book by more than `5bps`.
