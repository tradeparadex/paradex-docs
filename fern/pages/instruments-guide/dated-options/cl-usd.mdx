The reference configuration for the CL (Crude Oil) Dated Options contract:

| Parameter | Value |
|-----------|-------|
| Product type | Dated European Option |
| Symbol format | CL-USD-{Expiry}-{Strike}-{Type} |
| Base currency | CL |
| Quote currency | USD |
| Settlement currency | USDC |
| Price tick size | 0.001 USD |
| Order size increment | 1 CL |
| Minimum order value | 20 USD |
| Maximum order size | 10,000 CL |
| Maximum open orders | 150 |
| Position limit | 100,000 CL |
| Spot band factor | 50% |
| IV band factor | 50% |
