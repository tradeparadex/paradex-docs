# Authentication

## Account 2.0

Paradex uses Account 2.0 for authentication. To enable AI agent access, you'll need your **Paradex account private key**.

Your private key is used to sign requests and authorize trading actions on your behalf. Keep it secret and never share it publicly.

### Getting Your Private Key

1. Log in to [app.paradex.trade](https://app.paradex.trade)
2. Navigate to **Settings** → **API Keys**
3. Copy your account private key

### Using Your Key with AI Tools

Pass your private key via the `PARADEX_ACCOUNT_PRIVATE_KEY` environment variable when configuring the [MCP server](/agentic-ai-hub/mcp) or [CLI](/agentic-ai-hub/cli).

```bash
export PARADEX_ACCOUNT_PRIVATE_KEY=your_private_key
```

<Warning>
Never hardcode your private key in source code or commit it to version control. Use environment variables or a secrets manager.
</Warning>

### Read-Only vs. Trading Access

- **Read-only** (no key required): Market data, orderbook, funding rates, klines, system state
- **Trading** (key required): Account info, positions, orders, fills, vault management
