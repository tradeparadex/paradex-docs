---
title: BTC options configuration
---

The reference configuration for the BTC Dated Options contract:

| Parameter | Value |
|-----------|-------|
| Product type | Dated European Option |
| Symbol format | BTC-USD-{Expiry}-{Strike}-{Type} |
| Base currency | BTC |
| Quote currency | USD |
| Settlement currency | USDC |
| Price tick size | 0.01 USD |
| Order size increment | 0.001 BTC |
| Minimum order value | 20 USD |
| Maximum order size | 100 BTC |
| Maximum open orders | 150 |
| Position limit | 50 BTC |
| Spot band factor | 30% |
| IV band factor | 30% |
