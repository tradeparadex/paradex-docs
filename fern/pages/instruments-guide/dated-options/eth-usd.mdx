The reference configuration for the ETH Dated Options contract:

| Parameter | Value |
|-----------|-------|
| Product type | Dated European Option |
| Symbol format | ETH-USD-{Expiry}-{Strike}-{Type} |
| Base currency | ETH |
| Quote currency | USD |
| Settlement currency | USDC |
| Price tick size | 0.01 USD |
| Order size increment | 0.1 ETH |
| Minimum order value | 20 USD |
| Maximum order size | 1,000 ETH |
| Maximum open orders | 150 |
| Position limit | 10,000 ETH |
| Spot band factor | 40% |
| IV band factor | 40% |
