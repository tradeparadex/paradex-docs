### v1.122.5 - Nov 26th
üñ•Ô∏è UI Updates
* UI Refresh
  * Updated the `Color palette` to use red and blue, aligning with professional trading terminals and moving away from the traditional green/red scheme
  * Simplified the visual system using primary colors (red/blue) supported by Black and Gray tones to reduce noise and improve clarity.
  * Improved overall layout with cleaner spacing and reduced visual clutter by removing badges, underlines, and other nonessential elements.

* Layout & Information Density
  * Tightened spacing across the interface to present more information on a single screen and enhance the professional trading experience.

* Trading Information
  * Desktop: Reordered information rows for clearer logical grouping and visibility based on importance.
  * Mobile: Fully redesigned trading information layout to improve hierarchy and readability.
  * Key metrics such as `Entry Price`, `Liquidation Price`, and `Mark Price` are now grouped in a single row.
  * Less critical fields are deprioritised or hidden within informational cards to keep the view focused.
  * Position details such as `PnL` can now be viewed at a glance.

* Mobile Navigation
  * Added a chart icon to each card to provide immediate access to chart views.

* Mobile Trade Page
  * Extended the `Orderbook` display to show the same level of detail available on desktop.

* Input Improvements
  * Labels now float above input fields, allowing full-width usage while maintaining visibility of labels and values.
  * Desktop inputs are right-aligned for improved readability.
  * Labels are truncated when space is limited to preserve layout integrity.

* Quality of Life Enhancements
  * Enabled chart scrolling on mobile devices.
