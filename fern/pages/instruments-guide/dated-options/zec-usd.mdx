The reference configuration for the ZEC Dated Options contract:

| Parameter | Value |
|-----------|-------|
| Product type | Dated European Option |
| Symbol format | ZEC-USD-{Expiry}-{Strike}-{Type} |
| Base currency | ZEC |
| Quote currency | USD |
| Settlement currency | USDC |
| Price tick size | 0.001 USD |
| Order size increment | 0.1 ZEC |
| Minimum order value | 20 USD |
| Maximum order size | 2,000 ZEC |
| Maximum open orders | 150 |
| Position limit | 20,000 ZEC |
| Spot band factor | 50% |
| IV band factor | 50% |
