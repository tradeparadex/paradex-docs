Claude.ai web cannot run local programs. Use one of these options:

- **[Paradex-hosted MCP](/agentic-ai-hub/paradex-hosted-mcp)** — market data only, no install needed
- **[Remote deployment](/agentic-ai-hub/remote-mcp/quickstart)** — host your own MCP server, market data only

<Steps>
### Deploy or get a server URL

Follow one of the links above to get your MCP server URL.

### Connect in Claude.ai

1. Open [claude.ai](https://claude.ai) → **Settings → Integrations**.
2. Click **Add MCP Server**.
3. Enter your MCP server URL.
