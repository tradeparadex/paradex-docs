# ChatGPT Desktop

This page covers three independent setup options — choose one:

- **A.** [**Local MCP Setup**](#a-local-mcp-setup) — Full API access including trading. Requires installing a few tools on your machine.
- **B.** [**Remote MCP Deployment**](#b-remote-mcp-deployment) — Market data only. Host your own MCP server on a cloud platform.
- **C.** [**Paradex-hosted MCP**](#c-paradex-hosted-mcp) — Market data only. No installation needed, just add a URL.

---

## A. Local MCP Setup

<Note>
Local setup is the **only option that supports trading**.
</Note>

ChatGPT Desktop cannot run local programs directly, so you use ngrok to create a tunnel from your local MCP server to a public URL that ChatGPT can connect to.

### Step 1 — Prerequisites

Most steps require running commands in a terminal:

- **macOS:** Open Terminal — press **Cmd + Space**, type "Terminal", and press Enter. You can also find it in **Applications → Utilities → Terminal**.
- **Windows:** Right-click the Windows Start icon and select **Terminal**. Alternatively, search for "cmd", "PowerShell", or "Command Prompt" in the Start menu. Both PowerShell and Command Prompt will work for the commands in this guide.

#### Install Python

Python 3.10 or higher is required. Check your version in your terminal:

```bash
python3 --version
```

You should see something like `Python 3.12.4`. If the version is below 3.10 or the command is not found, install Python from [python.org/downloads](https://www.python.org/downloads/).

#### Install uvx

`uvx` is a fast Python package runner used to download and start the MCP server automatically.

```bash
pip install uv
```

On macOS you can also use Homebrew:

```bash
brew install uv
```

Verify the installation:

```bash
uvx --version
```

**Install ngrok:**

```bash
brew install ngrok   # macOS
# Or download from https://ngrok.com
```

<details>
<summary>Alternative: Run with Docker</summary>

If you prefer not to install Python or uvx directly, you can run the MCP server inside a Docker container. Requires Docker.

```bash
git clone https://github.com/tradeparadex/mcp-paradex-py.git
cd mcp-paradex-py
docker build . -t mcp-paradex-py

# Read-only mode (public market data only)
docker run --rm -i mcp-paradex-py

# With trading capabilities
docker run --rm -e PARADEX_ACCOUNT_PRIVATE_KEY=your_key -i mcp-paradex-py
```

</details>

### Step 2 — Get Your Paradex Private Key

All trading features require a Paradex account private key. Market data tools work without one — you can skip this step if you only need market data.

1. Go to [Paradex](https://app.paradex.trade) and connect your wallet.
2. Navigate to **Settings → API Keys** and create a new key.
3. Copy the private key — you'll use it below.

<Warning>
Your private key controls access to your trading account. Never share it publicly or commit it to version control. For more details, see [Authentication](/api/general-information/authentication).
</Warning>

<Note>
**Testnet first?** Use `PARADEX_ENVIRONMENT=testnet` and get test funds from the Paradex testnet faucet.
</Note>

---

### Step 3 — Configure

1. **Start the MCP server locally:**

   ```bash
   MCP_TRANSPORT=streamable-http MCP_PORT=8080 \
   PARADEX_ENVIRONMENT=prod \
   PARADEX_ACCOUNT_PRIVATE_KEY=your_private_key_here \
   uvx mcp-paradex
   ```

   Replace `your_private_key_here` with your actual key from Step 2.

2. **Open a tunnel** in a second terminal:

   ```bash
   ngrok http 8080
   ```

   Copy the `https://xxxx.ngrok-free.app` URL from the output.

3. **Connect in ChatGPT:**

   1. Open ChatGPT Desktop → **Settings → Advanced → Developer Mode** (enable if not already).
   2. Go to **Settings → Connectors → Create**.
   3. Paste `https://xxxx.ngrok-free.app/mcp`.
   4. Click **I trust this provider**.

<Note>
**Limitation:** The tunnel URL changes every time you restart ngrok (free tier). Update the connector URL each session.
</Note>

---

### Verify It's Working

Ask ChatGPT:

> "What markets are available on Paradex?"

If you get a list of trading pairs, you're all set. To test trading access:

> "Show me my Paradex account summary."

### Troubleshooting

**Tools not appearing**
- Confirm Developer Mode is enabled in ChatGPT settings
- Check that the connector URL ends with `/mcp`
- Make sure the local server and ngrok tunnel are both running

**Authentication errors**
- Double-check your private key is correct and starts with `0x`
- Verify `PARADEX_ENVIRONMENT` matches where your account was created (`prod` or `testnet`)

## B. Remote MCP Deployment

<Warning>
Remote setups do **not support trading** at the moment — only market data is available.
</Warning>

First, deploy the MCP server to a cloud platform by following the [Remote Deployment guide](/agentic-ai-hub/mcp/remote-deployment). Once you have your endpoint URL, return here to connect it to ChatGPT Desktop.

**Connect your endpoint:**

1. Open ChatGPT Desktop → **Settings → Advanced → Developer Mode** (enable if not already).
2. Go to **Settings → Connectors → Create**.
3. Enter your endpoint URL (e.g. `https://your-domain.example.com/mcp`).
4. Click **I trust this provider**.

## OpenAI Agents SDK (for developers)

For custom Python apps (not the ChatGPT interface). No remote hosting needed.

```bash
pip install openai-agents
```

```python
import asyncio, os
from agents import Agent, Runner
from agents.mcp import MCPServerStdio

async def main():
    async with MCPServerStdio(
        name="paradex",
        params={
            "command": "uvx",
            "args": ["mcp-paradex"],
            "env": {
                **os.environ,
                "PARADEX_ENVIRONMENT": "prod",
                "PARADEX_ACCOUNT_PRIVATE_KEY": "your_private_key_here",
            },
        },
    ) as server:
        agent = Agent(
            name="Paradex Trader",
            instructions="You are a trading assistant with access to Paradex.",
            mcp_servers=[server],
        )
        result = await Runner.run(agent, "What are the top markets by volume?")
        print(result.final_output)

asyncio.run(main())
```

### Verify It's Working

Ask ChatGPT:

> "What markets are available on Paradex?"

If you get a list of trading pairs, you're connected. Note that trading features are not available with remote setups.

## C. Paradex-hosted MCP

<Warning>
This setup does **not support trading** at the moment. Only public market data is available.
</Warning>

### Via URL

1. Open ChatGPT Desktop → **Settings → Advanced → Developer Mode** (enable if not already).
2. Go to **Settings → Connectors → Create**.
3. Enter `https://mcp.paradex.trade/mcp`.
4. Click **I trust this provider**.

### Via Smithery

1. Visit [smithery.ai](https://smithery.ai) and search for **Paradex**.
2. Select the Paradex MCP server.
3. Choose **ChatGPT** as your client.
4. Follow the guided setup and click **Connect**.

### Verify It's Working

Ask ChatGPT:

> "What markets are available on Paradex?"

If you get a list of trading pairs, you're connected. Note that trading features are not available with this setup.
