Paradex is a decentralized exchange (DEX) built as a Starknet appchain, combining the performance of centralized exchanges with the security and transparency of blockchain technology.

## Key Features

<CardGroup cols={3}>
  <Card title="Privacy" icon="fa-solid fa-eye-slash" href="/trading/privacy">
    Positions, entries, PnL and liquidation levels are hidden from the public.
  </Card>

  <Card title="Unified Margin" icon="fa-solid fa-layer-group" href="/risk/margin-system">
    Trade 90+ markets across Futures and Spot all in one account.
  </Card>
</CardGroup>
