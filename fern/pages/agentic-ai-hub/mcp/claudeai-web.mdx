Claude.ai web cannot run local programs. Use one of these options:

- **[Paradex-hosted MCP](/agentic-ai-hub/mcp/paradex-hosted-mcp)** — market data only, no install needed
- **[Remote deployment](/agentic-ai-hub/mcp/remote-mcp)** — host your own MCP server, market data only

After deploying remotely or getting the Paradex-hosted URL:

1. Open [claude.ai](https://claude.ai) → **Settings → Integrations**.
2. Click **Add MCP Server**.
3. Enter your MCP server URL.

---

## Verify it's working

Ask your AI assistant:

> "What markets are available on Paradex?"

If you get a list of trading pairs, you're connected.
