## Overview

Paradex spot markets let you buy and sell crypto assets directly. All markets are quoted in USD and settled in USDC at the prevailing USDC/USD rate.

<CardGroup cols={3}>
  <Card title="USD Quoted" icon="dollar-sign">
    All spot markets are priced in USD
  </Card>

  <Card title="USDC Settled" icon="money-bill-transfer">
    Trades settle in USDC using the prevailing USDC/USD price
  </Card>

  <Card title="USDC Fees" icon="receipt">
    All trading fees are charged in USDC
  </Card>
</CardGroup>

## Naming Convention

Spot markets use the format **`<BASE>-<QUOTE>`**, where the base currency is the asset being traded and the quote currency is used for pricing.

| Market Type | Symbol | Description |
| --- | --- | --- |
| Spot | `ETH-USD` | ETH priced in USD |
| Perpetual | `ETH-USD-PERP` | ETH perpetual contract |

<Note>
Even though markets are quoted in USD, all trades settle in USDC at the prevailing USDC/USD price.
</Note>

## Supported Order Types

The following order types are available for spot markets:

| Order Type | Supported |
| --- | --- |
| Market | Yes |
| Limit | Yes |
| IOC (Immediate-or-Cancel) | Yes |
| Post-Only | Yes |
| Reduce-Only | No |
| TP/SL | No |
