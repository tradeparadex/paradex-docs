# Prerequisites & Setup

Complete these steps before configuring any local AI client. If you're using Option C (Connect to Paradex MCP) or a remote-only setup, you can skip this page.

Most steps below require running commands in a terminal:

- **macOS:** Open Terminal — press **Cmd + Space**, type "Terminal", and press **Enter**. You can also find it in **Applications → Utilities → Terminal**.
- **Windows:** Right-click the Windows Start icon and select **Terminal**. Alternatively, search for "cmd", "PowerShell", or "Command Prompt" in the Start menu. Both PowerShell and Command Prompt will work for the commands in this guide.

## Install Python

Python 3.10 or higher is required. Check your version:

```bash
python3 --version
```

If you need to install or upgrade, visit [python.org/downloads](https://www.python.org/downloads/).

## Install uvx

`uvx` is a fast Python package runner used to download and start the MCP server automatically.

```bash
pip install uv
```

On macOS you can also use Homebrew:

```bash
brew install uv
```

Verify the installation:

```bash
uvx --version
```

## Alternative: Run with Docker

If you prefer not to install Python or `uvx` directly, you can run the MCP server inside a Docker container. Requires Docker installed on your machine.

```bash
# Clone the repository
git clone https://github.com/tradeparadex/mcp-paradex-py.git
cd mcp-paradex-py

# Build the image
docker build . -t mcp-paradex-py

# Run in read-only mode (public market data only)
docker run --rm -i mcp-paradex-py

# Run with trading capabilities
docker run --rm -e PARADEX_ACCOUNT_PRIVATE_KEY=your_key -i mcp-paradex-py
```

To use Docker with a specific AI client, replace `"command": "uvx"` in that client's config with the appropriate Docker run command. See each client's setup page for details.

## Get Your Paradex Private Key

All trading features require a Paradex account private key. Public market data tools work without one — you can skip this step if you only need market data.

1. Go to [Paradex](https://app.paradex.trade) and connect your wallet.
2. Navigate to **Settings → API Keys** and create a new key.
3. Copy the private key — you'll use it when configuring your AI client.

<Note>
**Testnet first?** Use `PARADEX_ENVIRONMENT=testnet` and get test funds from the Paradex testnet faucet.
</Note>

## Environment Variables Reference

These variables are used across all client configurations:

| Variable | Description | Default |
|---|---|---|
| `PARADEX_ENVIRONMENT` | Target environment: `prod`, `testnet`, or `nightly` | `prod` |
| `PARADEX_ACCOUNT_PRIVATE_KEY` | Your Paradex account private key (required for trading) | — |

If you've cloned the repository locally, you can store these in a `.env` file instead of passing them on the command line:

```bash
cp .env.template .env
```

Then edit `.env` with your values:

```
PARADEX_ENVIRONMENT=testnet
PARADEX_ACCOUNT_PRIVATE_KEY=0x_your_private_key_here
```

The server loads this file automatically on startup. Not needed if you set env in your client config — those take precedence.
