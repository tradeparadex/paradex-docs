## Buybacks

All net protocol revenue generated from fees across current and future products is expected to be allocated by Paradex to $DIME token buybacks and/or other ecosystem growth initiatives. Buybacks are typically executed periodically (for example, daily) using accumulated net protocol revenue.

The account used for buybacks is `0x3f38009a58bfc930a399ef9670ee60b355d9d22ea79709f2243182989d16709`. This mechanism is intended to reduce the circulating supply of $DIME over time.

## Staking

Staked $DIME is intended to provide access to platform benefits such as reduced fees, rebates, and priority access to select protocol features, including Gigavault. A full staking program will be announced shortly after TGE.

## Gas

$DIME is intended to be the gas token for Paradex Chain (DimeVM), a general-purpose smart contract platform powered by Cairo. DimeVM shares state with the Paradex Exchange, enabling a composable, high-throughput execution environment where onchain applications can interact with the liquidity and privacy infrastructure built on Paradex.
