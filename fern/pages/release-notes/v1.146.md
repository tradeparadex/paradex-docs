## Prod
### v1.146.2 - May 21st
🖥️ UI Updates
* Improved Mobile Menu UX.

⚙️ API Updates
* RPI orders are no longer restricted to non empty orderbooks.
* Added `GET /markets/settlement-price` endpoint to fetch details for settled markets.
* Added `GET /markets/history` endpoint to fetch all events related to markets, including creation, specs updates and settlement.
* New WS channel `markets` to receive instant notifications for all market-related activities (creation, specs updates, and settlement).
---
