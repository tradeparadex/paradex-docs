<Warning>
This setup does **not support trading** at the moment. Only public market data is available. For full API access including trading, use a [local setup](/agentic-ai-hub/mcp/quickstart).
</Warning>

Paradex hosts a public MCP server at `https://mcp.paradex.trade/mcp`. Connect any AI client that supports remote MCP servers — no installation, no API keys, no cloud deployment.

**What you can do:**

- Query live and historical market data
- Browse available trading pairs
- Check funding rates, order book depth, and more
