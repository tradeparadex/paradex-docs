The reference configuration for the HYPE Dated Options contract:

| Parameter | Value |
|-----------|-------|
| Product type | Dated European Option |
| Symbol format | HYPE-USD-{Expiry}-{Strike}-{Type} |
| Base currency | HYPE |
| Quote currency | USD |
| Settlement currency | USDC |
| Price tick size | 0.001 USD |
| Order size increment | 1 HYPE |
| Minimum order value | 20 USD |
| Maximum order size | 10,000 HYPE |
| Maximum open orders | 150 |
| Position limit | 100,000 HYPE |
| Spot band factor | 50% |
| IV band factor | 50% |
