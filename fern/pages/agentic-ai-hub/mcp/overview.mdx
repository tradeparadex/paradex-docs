# MCP Overview

**Repository:** [github.com/tradeparadex/mcp-paradex-py](https://github.com/tradeparadex/mcp-paradex-py)

The Paradex MCP server implements the [Model Context Protocol](https://modelcontextprotocol.io/) standard, giving AI assistants structured, authenticated access to the Paradex trading platform. Once configured, your AI assistant can query markets, manage positions, place orders, and monitor vaults — all through natural language.

## What the MCP Server Enables

- Fetch live and historical market data
- Manage trading accounts and vaults
- Place, monitor, and cancel orders
- Track positions, balances, and funding payments

## Pick Your Agent Client

Your setup depends on two things:

- **Which MCP features you need** — trading is only available with a local setup at the moment.
- **Which AI client you prefer** — Claude, Cursor, ChatGPT, etc.

### Option A — Local Setup (Recommended)

MCP server runs on your machine.

- Full API support including trading
- Requires prerequisites (Python 3.10+, uvx)
- Requires a Paradex private key

| Client | Notes |
|---|---|
| [**Claude Desktop**](/agentic-ai-hub/mcp/claude-desktop) (app) | Recommended. Best experience, full tool support |
| [**Claude Code**](/agentic-ai-hub/mcp/claude-code) (CLI) | One-line setup |
| [**Cursor**](/agentic-ai-hub/mcp/cursor) | One-click install available |
| [**VS Code (GitHub Copilot)**](/agentic-ai-hub/mcp/vscode) | Via `.vscode/mcp.json` |
| [**ChatGPT Desktop**](/agentic-ai-hub/mcp/chatgpt-desktop) (app) | Via ngrok tunnel |
| Other clients | [Windsurf, Gemini CLI, OpenAI Codex CLI](/agentic-ai-hub/mcp/other-clients) |

### Option B — Remote Deployment

Host your own MCP server endpoint on a cloud platform.

- No trading support at the moment
- Works with clients that cannot run local programs

| Client | Notes |
|---|---|
| [**ChatGPT Desktop**](/agentic-ai-hub/mcp/chatgpt-desktop) (app) | AWS Lambda, or permanent endpoint |
| [**Claude.ai**](/agentic-ai-hub/mcp/remote-deployment) (web) | Requires remote HTTP server |

See the [Remote Deployment guide](/agentic-ai-hub/mcp/remote-deployment) for platform options (Railway, Render, Fly.io, AWS Lambda, Docker).

### Option C — Connect to Paradex MCP

Connect directly to the hosted Paradex MCP server — no local install or cloud deployment needed.

- No trading support at the moment
- Easiest setup — just add a URL or use Smithery

**Smithery (recommended):** Visit [smithery.ai](https://smithery.ai) and search for Paradex, then follow the guided setup for your client.

**Direct URL:** Add the following MCP connector URL in your client's settings:

```
https://mcp.paradex.trade/mcp
```
