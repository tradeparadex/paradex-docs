## Prod
### v1.146.0 - May 19th
🖥️ UI Updates
* Add 'Managed By' curator to vaults list and page.
* Mobile menu redesign — tap your profile picture to access all menu items.
* The Terminal drawer can now be resized by dragging its left edge; the chosen width is remembered across sessions, with a max of 900px on screens wider than 1440px.
* Renamed the "Order Book" tab to "Book" for a more compact layout.

⚙️ API Updates
* Add optional curator and strategy fields on vaults.
