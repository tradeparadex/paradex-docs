## Buybacks

Net revenue generated from protocol fees will be used to buy back $DIME and/or fund ecosystem growth initiatives. Buybacks are executed periodically (e.g. Daily TWAP) using available net revenue.

The account that conducts buybacks is `0x3f38009a58bfc930a399ef9670ee60b355d9d22ea79709f2243182989d16709`

## Staking

Staked $DIME is intended to provide access to platform benefits such as reduced fees, rebates, and priority access to select protocol features, including Gigavault. A full staking program will be announced shortly.

## Gas

$DIME is intended to be the gas token for Paradex Chain (DimeVM), a general-purpose smart contract platform powered by Cairo. DimeVM shares state with the Paradex Exchange, enabling a composable, high-throughput execution environment where onchain applications can interact with the liquidity and privacy infrastructure built on Paradex.
