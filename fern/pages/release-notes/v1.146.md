## Prod
### v1.146.1 - May 20th
⚙️ API Updates
* Per-credential IP allowlist on subkeys and API keys — a credential can be locked to specific CIDR ranges so a leaked key cannot trade from elsewhere. Useful for isolating Hummingbot pods or other bots behind a static egress IP.

🖥️ UI Updates

* Mobile menu redesign — tap your profile picture to access all menu items.
* Renamed the "Order Book" tab to "Book" for a more compact layout.

⚙️ API Updates
* Add optional curator and strategy fields on vaults.
