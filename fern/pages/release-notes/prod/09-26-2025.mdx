## v1.113.12

<ChangelogTags tags={["UI", "API"]} />

* **UI:** View `lifetime fee savings` from zero-fee trading right in your `Portfolio` overview.
* **UI:** Users can create and manage their `trading-only` keys in new `Subkey` tab under Key Management settings.
* **UI:** `Order Book` retains your chosen tick size aggregation for each market.
* **UI:** `iOS` users should see numeric keyboard on numeric input field.
* **UI:** Mobile toast notifications can now be dismissed with a swipe.
* **UI:** Mobile toast notifications have been repositioned higher to avoid blocking the Order Builder.
* **API:** Users can now create and authenticate with Sub keys.
* **API:** Number of sub-accounts is now limited to 25 per account.
