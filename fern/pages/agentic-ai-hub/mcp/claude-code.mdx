# Claude Code CLI

<Note>
Complete the [Getting Started](/agentic-ai-hub/mcp/getting-started) steps (install Python, uvx, and get your private key) before continuing.
</Note>

## Local MCP Setup

<Note>
This step uses the Claude Code CLI. If you haven't installed it yet, run `npm install -g @anthropic-ai/claude-code` or see the [Claude Code docs](https://docs.anthropic.com/en/docs/claude-code) for other install methods.
</Note>

<Steps>
  <Step title="Add the Paradex MCP server">
    Run this command in your terminal. Replace `your_private_key_here` with your actual key from the Getting Started page:

    ```bash
    claude mcp add paradex uvx mcp-paradex \
      -e PARADEX_ENVIRONMENT=prod \
      -e PARADEX_ACCOUNT_PRIVATE_KEY=your_private_key_here
    ```
  </Step>
</Steps>

### Verify It's Working

Check that the server is registered:

```bash
claude mcp list
```

Then ask Claude Code:

> "What markets are available on Paradex?"

If you get a list of trading pairs, you're all set. To test trading access:

> "Show me my Paradex account summary."

### Troubleshooting

**Server not appearing in `mcp list`**
- Make sure `uvx` is installed: `uvx --version`
- Re-run the `claude mcp add` command

**`uvx: command not found`**
```bash
pip install uv
# or on macOS:
brew install uv
```

## Paradex-hosted MCP

<Warning>
This setup does **not support trading** at the moment. Only public market data is available.
</Warning>

### Via Smithery

<Steps>
  <Step title="Find the Paradex server">
    Visit [smithery.ai](https://smithery.ai) and search for **Paradex**.
  </Step>
  <Step title="Select and configure">
    Select the Paradex MCP server, choose **Claude Code** as your client.
  </Step>
  <Step title="Connect">
    Follow the guided setup and click **Connect**.
  </Step>
</Steps>

### Verify It's Working

Ask Claude Code:

> "What markets are available on Paradex?"

If you get a list of trading pairs, you're connected. Note that trading features are not available with this setup.
