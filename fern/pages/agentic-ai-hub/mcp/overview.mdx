# MCP Overview

**Repository:** [github.com/sv/mcp-paradex-py](https://github.com/sv/mcp-paradex-py)

The Paradex MCP server implements the [Model Context Protocol](https://modelcontextprotocol.io/) standard, giving AI assistants structured, authenticated access to the Paradex trading platform. Once configured, your AI assistant can query markets, manage positions, place orders, and monitor vaults — all through natural language.

## What the MCP Server Enables

- Fetch live and historical market data
- Manage trading accounts and vaults
- Place, monitor, and cancel orders
- Track positions, balances, and funding payments

## Prerequisites

- Python 3.10 or higher — check with `python3 --version`. If you need to install it, visit [python.org/downloads](https://www.python.org/downloads/).
