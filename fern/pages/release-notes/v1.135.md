## Prod
### v1.135.10 - March 6th
🖥️ UI Updates
* Vault strategies are now sorted alphabetically for easier navigation.
* Refined mobile TP/SL modals: fixed layouts, tabbed units, improved calculator UX, and resolved label clipping.
---
### v1.135.5 - March 5th
🖥️ UI Updates
* Internal transfers now display as `Transfer` instead of `Withdrawal` under `Transfers` tab for improved clarity.
* Revamped XP page with new season banners for more immersive experience.
* XP profile stats card redesigned to display stats beside the avatar in a cleaner, two‑column layout improving readability and visual balance. 
---
### v1.135.2 - March 4th
🖥️ UI Updates
* Fast update of liquidation price on position table.
* Order and Trade IDs in UI tables now display with a middle-ellipsis (e.g., `0x123...abc`).
---
### v1.135.1 - March 3rd
🖥️ UI Updates
* Improved responsiveness of leverage slider and styling changes.
* The `Key Management` section in the settings modal now displays a title, matching the presentation of other settings sections.
* Unauthenticated users can now view the Assets Balance on the Vaults page.
