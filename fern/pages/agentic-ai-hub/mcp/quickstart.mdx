## Step 2 — Pick Your Setup Path

### Option A — Local MCP Setup (Recommended)

MCP server runs on your machine.

- Full API support including trading
- Requires prerequisites (Python 3.10+, uvx) and a Paradex private key
- Steps are included on each client page

Click your client name in the table above to go to its setup page — each page includes prerequisites, private key setup, and configuration instructions.

### Option B — Remote MCP Deployment

Host your own MCP server endpoint on a cloud platform.

- No trading support at the moment — market data only
- Required for clients that cannot run local programs (Claude.ai web)
- Also available for ChatGPT Desktop as an alternative to local setup

See the [Remote Deployment guide](/agentic-ai-hub/mcp/remote-deployment) for platform options (Railway, Render, Fly.io, AWS Lambda, Docker) and client connection instructions.

### Option C — Connect to Paradex MCP

Connect directly to the hosted Paradex MCP server — no local install or cloud deployment needed.

- No trading support at the moment — market data only
- Easiest setup — just add a URL or use Smithery

Setup instructions are included on each client page under the "Connect to Paradex MCP" section. You can also see the [dedicated guide](/agentic-ai-hub/mcp/connect-to-paradex-mcp) for details.
