# Other Clients

This page covers Windsurf and Gemini CLI.

<Note>
Complete the [Getting Started](/agentic-ai-hub/mcp/getting-started) steps (install Python, uvx, and get your private key) before continuing.
</Note>

## Local MCP Setup

### Windsurf

<Steps>
  <Step title="Open MCP config">
    Open **Windsurf Settings → MCP Servers** and paste the following into `~/.codeium/windsurf/mcp_config.json`:

    ```json
    {
      "mcpServers": {
        "paradex": {
          "command": "uvx",
          "args": ["mcp-paradex"],
          "env": {
            "PARADEX_ENVIRONMENT": "prod",
            "PARADEX_ACCOUNT_PRIVATE_KEY": "your_private_key_here"
          }
        }
      }
    }
    ```
  </Step>
  <Step title="Restart Windsurf">
    Restart Windsurf to apply changes.
  </Step>
</Steps>

### Gemini CLI

<Note>
Requires the Gemini CLI. Install with `npm install -g @google/gemini-cli` or see the [Gemini CLI GitHub repo](https://github.com/google-gemini/gemini-cli) for details.
</Note>

<Steps>
  <Step title="Open settings file">
    Open (or create) the file `~/.gemini/settings.json` and paste the following:

    ```json
    {
      "mcpServers": {
        "paradex": {
          "command": "uvx",
          "args": ["mcp-paradex"],
          "env": {
            "PARADEX_ENVIRONMENT": "prod",
            "PARADEX_ACCOUNT_PRIVATE_KEY": "your_private_key_here"
          }
        }
      }
    }
    ```

    Replace `your_private_key_here` with your actual key from the Getting Started page.
  </Step>
</Steps>

### Verify It's Working

Ask your AI assistant:

> "What markets are available on Paradex?"

If you get a list of trading pairs, you're all set. To test trading access:

> "Show me my Paradex account summary."

### Troubleshooting

**Tools not appearing / MCP not loading**
- Make sure `uvx` is installed: `uvx --version`
- Check that the config file has valid JSON (no trailing commas)
- Restart the client after config changes

**`uvx: command not found`**
```bash
pip install uv
# or on macOS:
brew install uv
```

## Paradex-hosted MCP

<Warning>
This setup does **not support trading** at the moment. Only public market data is available.
</Warning>

### Via Smithery

<Steps>
  <Step title="Find the Paradex server">
    Visit [smithery.ai](https://smithery.ai) and search for **Paradex**.
  </Step>
  <Step title="Select and configure">
    Select the Paradex MCP server, choose your client from the list.
  </Step>
  <Step title="Connect">
    Follow the guided setup and click **Connect**.
  </Step>
</Steps>

### Verify It's Working

Ask your AI assistant:

> "What markets are available on Paradex?"

If you get a list of trading pairs, you're connected. Note that trading features are not available with this setup.
